# LCDsystem(这是一个lcd显示屏的app)

大致功能如下：
完成开机启动APP,自动播放视频或广告，通过tcp连接生成二维码供车辆扫码付费放行。系统可以设置时间，声音的大小，亮度；还可以定时设置每天开关机的时间。
